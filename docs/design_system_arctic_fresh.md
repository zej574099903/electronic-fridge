# Arctic Fresh 设计系统文档

`Arctic Fresh` 是为“电子冰箱”移动应用量身定制的顶级视觉设计系统。该系统旨在将基础的库存管理工具提升为极具高级感、专业感且操作顺滑的旗舰级产品。

## 1. 设计理念
- **简约而不简单**：通过极简的线条和深邃的配色，营造专业可靠的氛围。
- **深度感 (Depth)**：利用多层阴影和微弱的边框渐变，让界面呈现出立体的悬浮层级。
- **呼吸感 (Breathability)**：通过大圆角和充裕的内边距，减少操作压力。

## 2. 色彩系统 (Color Palette)
系统采用名为“极地新鲜”的高级色调：

| 角色 | 颜色值 | 描述 |
| :--- | :--- | :--- |
| **Primary (主色)** | `#1E293B` | 深海军蓝，用于标题、主按钮，给人稳重、高端的初印象。 |
| **Secondary (次色)** | `#3B82F6` | 现代感冰蓝色，用于交互反馈、图标和次要强调。 |
| **Accent (强调色)** | `#F97316` | 活力橙，用于临期提醒，提供强烈的视觉警示。 |
| **Surface (表面色)** | `#FFFFFF` | 纯白卡片背景。 |
| **Surface Alt (辅助表面)** | `#F1F5F9` | 低饱和度灰蓝色，用于背景和输入框，增加层次感。 |
| **Text Primary** | `#0F172A` | 深度黑，确保最高级的阅读体验。 |

## 3. UI 组件规范

### 3.1 容器 (ScreenContainer)
- 全局使用 `#F8FAFC` 作为底色，与纯白卡片形成微妙对比，消除纯白背景的廉价感。

### 3.2 卡片 (SectionCard)
- **圆角**: `24pt` 大圆角，呈现现代移动端美学。
- **阴影**:
  - Android: `elevation: 4`
  - iOS: `shadowOpacity: 0.1, shadowRadius: 15`
- **边框**: `1pt` 的 `#E2E8F0` 细边框，辅助增强轮廓感。

### 3.3 交互元素
- **按钮**: 高度统一为 `56-64pt`，圆角 `20pt`。主按钮带主色同色系的投影 (`shadowOpacity: 0.2`)。
- **输入框**: 背景使用 `surfaceAlt`，带有 `1.5pt` 的精致边框。

## 4. 图标系统
- **库**: `@expo/vector-icons (Ionicons)`
- **风格**: 统一使用 `Outline` 风格图标，线条粗细与文字排版相匹配，提升整体精致度。
